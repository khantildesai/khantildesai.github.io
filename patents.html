Methods, systems and computer-readable media for training document type prediction models, and use thereof for creating accounting records

During my time as an intern at Xero, I worked on an accounting document type classifier. During my time on this project I ideated and created a ML model architecture and training pipeline that uses multimodal input and an innovative approach to a GRU to create a state-of-the-art solution.
